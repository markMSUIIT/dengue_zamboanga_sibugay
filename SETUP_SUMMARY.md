## ğŸ“ Troubleshooting

### "Module not found" Error
```bash
pip install --upgrade -r requirements.txt
```

### "Data file not found" Error
Ensure `sibugay_dengue_cases_dataset.csv` is in the same directory as `app.py`

1. Check CSV file has all required columns
2. Verify data types (numeric values, not text)
3. Run: `python3 verify_setup.py`

### App runs slowly
- Reduce year range in filter
- Select fewer municipalities
- Close other browser tabs

### Need to stop the dashboard
Press `Ctrl+C` in terminal
